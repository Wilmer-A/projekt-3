Om oss

Wilmer's Att-göra-listor är en webbsida där du kan anteckna saker som du måste göra för att inte glömma bort att göra det.

Webbsidan används på följande sätt:

  • Använd textrutan för att skriva något och tryck på 'Enter' eller plus-ikonen för att lägga till det i din lista
  • Tryck på kryss-ikonen för att ta bort en punkt från listan
  • Tryck på utropstecken-ikonen för att understryka en punkt som är extra viktig
  • Tryck på bock-ikonen för att markera att du är klar med en punkt
